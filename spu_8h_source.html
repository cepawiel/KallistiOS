106  upon completion (in an interrupt context!). Returns <0 on error. */

111 /** \brief Copy a block of data from SH4 RAM to sound RAM via DMA.
112 
113  This function sets up a DMA transfer from main RAM to the sound RAM with G2
114  DMA.
115 
116  \param from A pointer in main RAM to transfer from. Must be
117  32-byte aligned.
118  \param dest Offset in sound RAM to transfer to. Do not include
119  the 0xA0800000 part, its implied. Must be 32-byte
120  aligned.
121  \param length Number of bytes to copy. Must be a multiple of 32.
122  \param block 1 if you want to wait for the transfer to complete,
123  0 otherwise (use the callback for this case).
124  \param callback Function to call when the DMA completes. Can be NULL
125  if you don't want to have a callback. This will be
126  called in an interrupt context, so keep that in mind
127  when writing the function.
128  \param cbdata Data to pass to the callback function.
129  \retval -1 On failure. Sets errno as appropriate.
130  \retval 0 On success.
131 
132  \par Error Conditions:
133  \em EINVAL - Invalid channel \n
134  \em EFAULT - from or dest is not aligned \n
135  \em EIO - I/O error
136 */
137 int spu_dma_transfer(void *from, uintptr_t dest, size_t length, int block,
138  spu_dma_callback_t callback, void *cbdata);
